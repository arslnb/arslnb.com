Manifest and Install Errors

ErrorLikely CauseFix
unsupported manifest schema_versionWrong schema version.Set schema_version to 1.
manifest id is requiredMissing plugin id.Add non-empty id.
components.client.entry is requiredClient component missing entry path.Provide valid components.client.entry.
server entry not foundServer entry path invalid.Fix components.server.entry relative path.
artifact checksum mismatchRegistry metadata mismatch or tampered artifact.Re-publish artifact and metadata with correct SHA256/signature.

OAuth Errors

ErrorCauseFix
oauth provider not foundProvider id not declared in manifest.Add provider under oauth.providers with matching id.
invalid redirect_uriRedirect URI mismatch with OAuth app registration.Update provider app config to exactly match Zeus callback URI.
token exchange failedBad client credentials, code, or scope mismatch.Verify client id/secret refs and authorization request params.

Runtime and Hook Issues

  • before_tool_call with blocked=true stops execution by design.
  • before_model_resolve and before_prompt_build ignore block flags.
  • If handler is not invoked, check operation name matches manifest declaration and policy enables it.
  • Verify handler signature supports expected call style (payload, context).

Channel and Webhook Issues

  1. Check plugin status via GET /v1/client/plugins.
  2. Verify channel name and route name normalization (lowercase + underscore/hyphen policy used by your plugin).
  3. Confirm provider callback URL matches /v1/channels/{channel}/webhook or /webhook/{route}.
  4. Ensure ingest_webhook returns a list of valid envelope dicts.

Quick Validation Checklist

  • Manifest passes validation and referenced entry files exist.
  • Required config and secret refs are present.
  • Local plugin appears in /api/plugins.
  • Server plugin appears in /v1/client/plugins (if server component exists).
  • At least one runtime call path succeeds (tool, send route, or webhook route).
If still blocked, capture exact request payload + response body for the failing endpoint.
