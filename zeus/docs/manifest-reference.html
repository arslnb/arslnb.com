Manifest Reference

Complete contract for zeus.plugin.json (Zeus Native Plugins).

Required Top-Level Fields

  • schema_version: must be 1
  • id: plugin id (slug-safe, stable)
  • version: plugin version string
  • components: at least one of client or server

Components Contract

FieldTypeRules
components.client.entrystringRequired when client component exists. Relative path to Python entry module.
components.client.runtimestringMust be python.
components.server.entrystringRequired when server component exists. Relative path to Python entry module.
components.server.runtimestringMust be python.
components.server.channelsarray<string>Channel names provided by server plugin.

Daemon Operation and Hook Fields

FieldTypePurpose
toolsarray<string|object>Model-callable operations.
commandsarray<string>Command operations.
servicesarray<string>Service operations.
cliarray<string>CLI operations.
providersarray<string>Provider operations.
gatewayarray<string>Gateway operations.
httparray<string>HTTP operations.
hook_phasesarray<string>Any of before_model_resolve, before_prompt_build, before_tool_call, after_tool_call.

Runtime Policy

FieldTypePurpose
runtime_policy.network.socat_bypassbooleanDefault network gate posture for plugin operations.
runtime_policy.approval.modestringask_each_time or allow_without_approval.
runtime_policy.tools.{name}.enabledbooleanEnable/disable specific tool.
runtime_policy.tools.{name}.allow_networkbooleanAllow network for that tool.
runtime_policy.tools.{name}.allow_without_approvalbooleanPermit execution without approval gate.
prerequisites.cli[]array<object>Declare required/optional CLI dependencies and install guidance.
prerequisites.environment_check.client_functionstringModule function that returns readiness ok/message.

OAuth Declaration

Use oauth.providers to register generic OAuth providers that Zeus can drive with shared flows.

FieldTypeRules
idstringUnique provider id within plugin.
componentstringclient or server (default server).
authorization_urlstringHTTPS URL for consent start.
token_urlstringHTTPS URL for token exchange/refresh.
scopesarray<string>Requested scopes.
pkcestringrequired, optional, or disabled.
client_id_fieldstringMust reference a declared config field.
client_secret_fieldstringMust reference a declared secret field (secret: true).
supports_refreshbooleanWhether refresh flow is supported.

Behavior, Scheduled Task, and Proactive Fields

FieldTypePurpose
automations.behaviors[]array<object>Install-time behavior prompts.
automations.scheduled_tasks[]array<object>Install-time scheduled prompts/cron-style jobs.
proactive.screen_watch[]array<object>Prompts consumed by proactive screen system.

Config Schema Rules

  • Use component-scoped schemas under config_schema.client and/or config_schema.server.
  • Each field must declare type and boolean secret.
  • Allowed type: string, integer, number, boolean, object, array.
  • secret: true values must be supplied via secret_refs, not plain config.
  • Unknown config keys are rejected.
