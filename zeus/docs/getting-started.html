Getting Started

Bare minimum files, manifest structure, and first install for Zeus plugins.

1) Pick Your Plugin Shape

ShapeManifest ComponentUse Case
Daemon-onlycomponents.clientExpose runtime tools/hooks inside local Zeus daemon.
Channel-onlycomponents.serverExpose channel webhooks/send or custom webhook routes on server.
Hybridcomponents.client + components.serverOne plugin bundle for both local and server behavior.

2) Bare Minimum Files

Daemon-only

my_daemon_plugin/
  zeus.plugin.json
  client/
    plugin.py

Channel-only

my_channel_plugin/
  zeus.plugin.json
  server/
    plugin.py

Hybrid

my_hybrid_plugin/
  zeus.plugin.json
  client/
    plugin.py
  server/
    plugin.py

3) Bare Minimum Manifest

{
  "schema_version": 1,
  "id": "acme_demo",
  "name": "Acme Demo",
  "version": "1.0.0",
  "components": {
    "client": { "entry": "client/plugin.py", "runtime": "python" }
  },
  "config_schema": {
    "client": {
      "api_base": { "type": "string", "required": true, "secret": false }
    }
  }
}

For channel plugins, replace/add components.server with entry, runtime, and channels.

4) Minimal Runtime Code

Daemon plugin entry

class DemoPlugin:
    def __init__(self, ctx):
        self.ctx = ctx

    def tools(self):
        return {
            "demo_tool": {
                "description": "Simple demo tool",
                "handler": self.demo_tool,
            }
        }

    async def demo_tool(self, payload, context):
        return {"ok": True, "echo": payload}


def register(context):
    return DemoPlugin(context)

Channel plugin entry

class DemoAdapter:
    def ingest_webhook(self, *, request, body):
        return [{"message_id": "demo:1", "channel": "demo", "text": "hello"}]

    def send(self, payload):
        return {"ok": True, "provider_message_id": "123"}


class DemoChannelPlugin:
    def __init__(self, ctx):
        self.ctx = ctx

    def channels(self):
        return {"demo": DemoAdapter()}


def register(ctx):
    return DemoChannelPlugin(ctx)

5) Add Common Features (Examples)

Tool + policy controls

{
  "tools": [
    { "name": "lookup_customer", "enabled": true, "allow_network": true, "allow_without_approval": false }
  ],
  "runtime_policy": {
    "network": { "socat_bypass": false },
    "approval": { "mode": "ask_each_time" },
    "tools": {
      "lookup_customer": {
        "enabled": true,
        "allow_network": true,
        "allow_without_approval": false
      }
    }
  }
}

CLI prerequisites + environment check

{
  "prerequisites": {
    "cli": [
      {
        "name": "GitHub CLI",
        "binary": "gh",
        "required": true,
        "install_hint": "brew install gh",
        "install_url": "https://cli.github.com/"
      }
    ],
    "environment_check": {
      "client_function": "check_environment"
    }
  }
}

# client/plugin.py
import shutil

def check_environment(context):
    if not shutil.which("gh"):
        return False, "please install gh with brew install gh"
    return True, ""

Hooks + operation kinds (commands/services/cli/providers)

{
  "commands": ["sync_customers"],
  "services": ["customers_service"],
  "cli": ["customers_cli"],
  "providers": ["customers_provider"],
  "gateway": ["customers_gateway"],
  "http": ["customers_http"],
  "hook_phases": ["before_prompt_build", "before_tool_call", "after_tool_call"]
}

OAuth provider declaration

{
  "oauth": {
    "providers": [
      {
        "id": "google",
        "component": "server",
        "authorization_url": "https://accounts.google.com/o/oauth2/v2/auth",
        "token_url": "https://oauth2.googleapis.com/token",
        "scopes": ["openid", "email"],
        "pkce": "required",
        "client_id_field": "google_client_id",
        "client_secret_field": "google_client_secret",
        "supports_refresh": true
      }
    ]
  }
}

Custom webhook route from plugin

def webhooks(self):
    return {
      "stripe-events": {
        "methods": ["POST"],
        "handler": self.handle_stripe
      }
    }

# Exposed as: POST /webhook/stripe-events

6) Install + Configure

Local install

POST /api/plugins/install
{
  "source_type": "path",
  "source": "/absolute/path/to/my_plugin",
  "force": true
}

Set config + secret refs

POST /api/plugins/config
{
  "plugin_id": "acme_demo",
  "config": {
    "client": { "api_base": "https://api.acme.dev" },
    "server": {}
  },
  "secret_refs": {
    "client": { "api_key": "acme_api_key" },
    "server": {}
  }
}

Server install (for server component)

POST /v1/client/plugins/install
{
  "plugin_id": "acme_demo",
  "version": "1.0.0",
  "config": { "server": {} },
  "secret_refs": { "server": {} }
}

7) Validate It Works

  • GET /api/plugins shows local plugin as installed/enabled.
  • GET /v1/client/plugins shows server plugin status (if server component exists).
  • Tool appears in daemon runtime and can be invoked.
  • Channel route and/or custom webhook route responds as expected.
